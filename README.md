# Moving Ball
In this game I created a ball movement simulator.

The user can adjust:

      •	Size of the ball.

      •	Velocity of the ball.

      •	Movement type.

      •	Background and ball color.

      •	Add or remove balls.

      •	 Stop or start the motion.
